# TheJudge
A tool for AI-response judgement and choice 

## Setup and installation
Since the project is not finished, it has to be run with an IDE.
### Run with Intellij IDEA
Maven is required
- Clone the repo inside an empty directory
- Open it with the IDE
- Right-click on the file pom.xml (path: Lumena/pom.xml) and select "Add as Maven Project"
- Go to File > Project Structure > Project and select OpenJDK version 17 as SDK, select 17 as language level
- Run the HelloApplication class (path: Lumena/src/main/java/com/skbd/simulatore/view/HelloApplication.java)
